Physics Notes

                    

                        

                            
Mass Formula:

                            

                                
M = √(n/α') * √T * √(D/10)

                            

                            
    
                                
  • n: energy level (0 = ground state)
    • 
                                
  • α': fundamental string length scale
    • 
                                
  • T: string tension
    • 
                                
  • D: number of dimensions
    • 
                            

                        


                        

                            
Key Points:

                            
    
                                
  • Ground state (n=0) stays at zero mass
    • 
                                
  • Higher states scale with √tension
    • 
                                
  • 10D is special (superstring theory)
    • 
                                
  • Extra dimensions are rolled up very small
